## ✨ Features

- 🌌 **Animated Starfield** — 200 rotating stars rendered on HTML5 Canvas
- 🌠 **Shooting Stars** — periodic shooting star animation
- 🌈 **Aurora Background** — 5-band animated aurora borealis effect
- 🗂️ **Multi-View Navigation** — smooth SPA-style navigation between pages (no reload)
- 📋 **One-Click Copy** — copy code snippets with a single tap
- 📁 **Dynamic File List** — folder system with categorized file listings
- 💎 **Glassmorphism UI** — modern frosted-glass card design
- 📱 **Fully Responsive** — optimized for mobile and desktop
- 🔒 **Source Protection** — JS obfuscated + HTML minified for deployment

## 🚀 Deployment

This project is deployed on **Vercel** as a static site.

### Deploy via Vercel Dashboard

1. Clone or download this repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **"Add New Project"** → Import this repository
4. Click **Deploy** — no build configuration needed

### Deploy via Drag & Drop

1. Go to [vercel.com/new](https://vercel.com/new)
2. Drag and drop the project folder
3. Done ✅
