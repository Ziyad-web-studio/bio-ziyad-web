Buatkan satu file HTML yang berisi aplikasi web chat AI bernama "Aiminizer". Gunakan Tailwind CSS (via CDN) dan JavaScript murni. Desain harus minimalis, terinspirasi dari UI Apple, menggunakan efek glassmorphism, dan memiliki mode gelap terang (dukungan warna CSS variables). Palet warna utama adalah hitam pekat, abu-abu, dan aksen oranye terang. Sertakan fitur dan struktur berikut: 1. Elemen Visual Latar: Tambahkan animasi bola cahaya (orb) yang berdenyut lambat di latar belakang. 2. Header (Efek Kaca): Melayang di atas. Kiri untuk tombol buka pengaturan. Tengah untuk judul "Aiminizer" dan teks kecil "ziyad_studio". Kanan untuk dropdown pilihan model (GPT-3.5 Turbo dan DeepSeek). 3. Area Obrolan: Bisa di-scroll. Pesan pengguna berwarna oranye. Pesan AI berwarna latar sesuai tema. Tampilkan animasi 3 titik memantul saat AI memproses jawaban. 4. Input Pesan (Efek Kaca): Melayang di bawah. Textarea yang tingginya otomatis menyesuaikan teks dan tombol kirim. 5. Panel Pengaturan: Muncul dari samping. Berisi tombol ubah mode gelap, input password API Key, dropdown Base URL (ChatAnywhere), dan textarea untuk System Prompt. Semuanya disimpan di LocalStorage. 6. Pemrosesan Teks: Gunakan library marked.js via CDN agar pesan AI mendukung Markdown (tabel, kode, daftar). Tambahkan efek animasi mesin tik saat AI merespons. Munculkan tombol salin di setiap gelembung pesan AI. 7. Logika Komunikasi: Gunakan Fetch API. Gabungkan system prompt bawaan pengguna dengan prompt tersembunyi absolut yang menyatakan pembuatnya adalah "Ziyad (ziyad_studio), individu muda dengan ambisi ekstrem dan pemikiran strategis". Kodenya harus rapi, responsif untuk perangkat seluler, dan siap dijalankan langsung di browser.
